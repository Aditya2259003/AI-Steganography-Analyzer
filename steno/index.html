Analysis Pipeline
// How the StegoSec workflow is organized
1. Encode: image is loaded into a hidden canvas, message bytes are converted to bits, and the least significant bits of RGB channels are updated.

2. Decode: the first 32 bits store payload length, then the app reconstructs the hidden message from subsequent LSB values.

3. Detect: the app extracts grayscale statistics, entropy, chi-square anomaly, LSB ratio, block entropy, and edge density to estimate whether an image looks suspicious.
